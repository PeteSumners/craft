## Tips for Using Presets

### 1. Copy-Paste in Blocks
Copy 5-10 commands at a time, paste into game.

### 2. Mark Your Progress
Place a colored wool block after each section you complete.

### 3. Fly Mode
Press Tab to toggle flying - essential for large projects.

### 4. Use Teleport
```
/pq 0 0     # Return to starting chunk
/spawn      # Go to spawn
```

### 5. Take Breaks
Rendering is I/O intensive. Take breaks between large sections.

### 6. Adjust for Your World
All coordinates assume starting at (0, Y, 0). Adjust X and Z as needed for your world.

## Custom Layout Calculator

**Formula for grid positioning**:

```
X position = column_index × (column_width + spacing)
Z position = row_index × (row_depth + spacing)
Y position = constant (e.g., 200)

Example:
- Column width: 350 voxels
- Spacing: 50 voxels
- Total: 400 voxels per column

Column 0: X = 0
Column 1: X = 400
Column 2: X = 800
Column 3: X = 1200
```

## Saving Your Work

The game auto-saves every 5 seconds. Your Bible monuments are permanent!

To backup your world:
1. Exit game
2. Copy `Craft/craft.db` to a safe location
3. That's your entire world including all rendered text
